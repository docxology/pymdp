| Module | Role |
|--------|------|
| [support/bootstrap.py](support/bootstrap.py) | Repo path / import setup |
| [support/mirror_dispatch.py](support/mirror_dispatch.py) | Dispatch to real pymdp APIs |
| [support/checks.py](support/checks.py), [support/patterns.py](support/patterns.py), [support/report.py](support/report.py), [support/viz.py](support/viz.py), [support/si_fixtures.py](support/si_fixtures.py) | Shared helpers for orchestrations |

Install context: editable `inferactively-pymdp` from parent directory ([docxology/pyproject.toml](../pyproject.toml)).
